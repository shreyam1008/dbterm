Keyboard-first DB client

dbterm

One tiny binary to work with PostgreSQL, MySQL, and SQLite without Java, Python, or Node runtimes.

Download Latest GitHub Repo
  • ~10MB RAM idle
  • Cross-platform
  • Single binary

Quick install

Install in one command. No Go toolchain required.

Linux / macOS

curl -fsSL https://raw.githubusercontent.com/shreyam1008/dbterm/main/install.sh | bash

Windows (PowerShell)

powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/shreyam1008/dbterm/main/install.ps1 | iex"

Ways to use dbterm

Ad-hoc SQL checks

Jump into any connection, run quick diagnostics, inspect rows, and sort results.

Schema exploration

Browse tables fast and use the built-in cheatsheets for PostgreSQL, MySQL, and SQLite.

Ops workflow

Use service dashboard shortcuts, run backups (PG/MySQL), and refresh DB state quickly.

Command examples

Most common commands in day-to-day usage:

dbterm
dbterm --help
dbterm --version
dbterm --info
dbterm --update
dbterm --update 0.3.4
dbterm --uninstall --yes
dbterm --uninstall --purge

Essential shortcuts

Shortcut Action
Alt + Enter Execute query
Alt + H Open help and SQL cheatsheets
Alt + Q / T / R Focus query, tables, or results
F5 / Ctrl + F5 Refresh current table or full table list
Ctrl + C Quit
